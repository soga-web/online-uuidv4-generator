# UUID v4 Generator

ブラウザ上で UUID v4 を生成し、ワンクリックでコピーできるシンプルな静的サイトです。

## Files

- `dist/index.html`
- `dist/styles.css`
- `dist/script.js`
- `dist/favicon.svg`
- `dist/robots.txt`

## Usage

`dist/index.html` をブラウザで開いて利用します。

ローカルサーバーで確認する場合は、dist ディレクトリで次を実行してください。

```bash
python3 -m http.server 8000
```

## Features

- UUID v4 の生成
- クリップボードへのコピー
- コピー成功時のボタン表示切り替え
- 再生成アクション
